# 系统设计说明书

本项目按“游客交互端 + 管理后台 + 后端 API + AI 服务层 + 数据层”建设。

## 核心模块

1. 游客端：数字人问答、语音输入、地图导览、路线推荐。
2. 管理后台：知识库管理、数字人配置、游客感受度报告、数据大屏。
3. 后端 API：会话、问答编排、景点、路线、后台统计。
4. AI 服务层：RAG、Prompt、模型适配、ASR/TTS、情感分析。
5. 数据层：PostgreSQL、Redis、向量库和文件存储。

## MVP 闭环

游客提问 -> RAG 检索 -> Mock/模型回答 -> 返回语音地址、表情、引用来源 -> 前端数字人播报和口型动画 -> 交互数据进入后台大屏。
